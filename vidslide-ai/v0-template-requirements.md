# V0 模板系统UI设计需求文档

## 🎯 项目概述

VidSlide AI 是一款将视频自动转换为专业PPT演示的工具。本文档为V0设计模板系统的UI界面。

## 🍎 设计指导 - 苹果设计师的思路

**核心理念**: 请以苹果设计师的身份来思考和设计这个模板系统。参考以下苹果设计原则：

### 设计哲学 (Design Philosophy)

- **Clarity Above All (清晰至上)**: 让每个元素的目的都清晰明了
- **Content is King (内容为王)**: 设计服务于内容，而不是喧宾夺主
- **Attention to Detail (注重细节)**: 每一个像素、每一个动画都要精心雕琢
- **Elegant Simplicity (优雅简洁)**: 用最少的元素表达最丰富的内涵

### 苹果设计师的思考方式

1. **用户视角**: 站在用户角度思考，每一个设计决策都要服务用户目标
2. **系统一致性**: 确保所有模板都遵循相同的视觉语言和交互模式
3. **性能意识**: 动画流畅、响应迅速，绝不容忍卡顿
4. **情感连接**: 设计不仅仅是功能，更是情感体验

### 参考标准

- **iOS界面设计**: 圆角、留白、层次感
- **Keynote动画**: 优雅的过渡和效果
- **macOS设计语言**: 毛玻璃效果、深度感
- **Apple Watch表盘**: 信息密度与美学的完美平衡

## 📋 核心设计原则 - 以苹果设计师的思路

**重要提示**: 请以苹果设计师的视角来设计这个模板系统。参考iOS界面设计、macOS设计语言，以及苹果的整体设计哲学：

### 🍎 苹果设计核心原则 (Apple Design Principles)

- **Clarity (清晰)**: 让内容易于理解，减少认知负担
- **Depth (深度)**: 通过层次、阴影和透视创造视觉深度
- **Deference (谦逊)**: 让内容成为焦点，界面元素退居其次
- **Fluidity (流畅)**: 自然的动画和过渡，让交互感觉无缝
- **Consistency (一致)**: 统一的语言和行为模式
- **Metaphors (隐喻)**: 使用熟悉的视觉元素和交互模式

### 🎨 设计风格定位

- **极简主义**: 去除一切不必要的元素
- **优雅简洁**: 参考iOS的界面设计风格
- **专业品质**: 达到苹果 Keynote 或 Pages 的设计水准
- **用户中心**: 以用户体验为首要考虑
- **现代化**: 使用最新的设计趋势和苹果的设计语言

## 🎨 模板类型及UI设计

### 1. 画中画模板 (Picture-in-Picture)

**适用场景**: 演讲者讲解时，自动将视频转为画中画显示文字/图表内容

#### UI层级结构

```
背景遮罩层 (固定)
├── 半透明黑色遮罩 (40%不透明度)
├── 画中画容器 (圆形/圆角矩形)
│   ├── 主视频画面 (实时缩放)
│   ├── 边框样式 (2px白色边框)
│   └── 位置选择 (右上/右下/左上/左下)
└── 内容展示区 (全屏)
    ├── 文字卡片/图表
    └── 渐入动画效果
```

#### 设计参数 - 苹果风格优化

- **画中画尺寸**: 28%主画面 (更优雅的比例)
- **圆角**: 16px (iOS风格圆角)
- **阴影**: 柔和的投影 (blur: 20px, opacity: 0.15)
- **边框**: 极细的白色边框 (0.5px, opacity: 0.8)
- **动画**: 苹果风格的缓动函数 (ease-out, 0.3s)
- **材质感**: 亚克力效果背景 (类似macOS的毛玻璃效果)
- **位置选择**: 智能推荐位置，避免遮挡重要内容

### 2. 信息卡片模板 (Info Card)

**适用场景**: 结构化展示要点，适合知识类内容

#### UI层级结构

```
背景遮罩层 (固定)
├── 深色半透明遮罩 (60%不透明度)
├── 卡片容器 (金色边框)
│   ├── 标题区域 (白色文字)
│   ├── 内容行列表 (1-3行)
│   │   ├── 彩色方块 (30%宽度)
│   │   ├── 箭头图标 (金色)
│   │   └── 文字内容 (可编辑)
│   └── 交互高亮 (脉冲效果)
└── 动画效果 (中心放大0.3s)
```

#### 设计参数 - 苹果风格优化

- **卡片尺寸**: 85%×65%屏幕 (更大的留白)
- **圆角**: 24px (更大的圆角，iOS风格)
- **边框**: 极细的金色边框 (1px, #FFD700, opacity: 0.6)
- **背景**: 毛玻璃效果 (blur backdrop, 类似iOS通知中心)
- **阴影**: 柔和多层阴影 (blur: 40px, y: 20px, opacity: 0.1)
- **间距**: 更大的元素间距 (24px系统)
- **层次感**: 使用z-index创造深度感

### 3. 关键词高亮模板 (Keyword Highlight)

**适用场景**: 强调核心概念，适合演讲重点突出

#### UI层级结构

```
背景遮罩层 (固定)
├── 浅色遮罩 (45%不透明度)
├── 高亮卡片 (居中偏上)
│   ├── 金色边框 (2px)
│   ├── 主标题 (大号白色)
│   ├── 副标题 (英文翻译)
│   └── 文字阴影效果
└── 动画 (淡入0.2s + 上浮10px)
```

#### 设计参数 - 苹果风格优化

- **卡片尺寸**: 60%×22%屏幕 (更大的比例，更好的视觉平衡)
- **圆角**: 20px (优雅的圆角)
- **背景**: 半透明白色 (opacity: 0.95, blur: 20px)
- **边框**: 极细的金色描边 (1px, #FFD700)
- **阴影**: 多层柔和阴影 (创造浮动效果)
- **文字**: SF Pro Display字体风格 (优雅的字间距)
- **动画**: 苹果的弹出动画 (scale + fade, spring效果)

### 4. 文件展示模板 (Document Display)

**适用场景**: 展示图表、报告等文档内容

#### UI层级结构

```
背景遮罩层 (固定)
├── 深色遮罩 (65%不透明度)
├── 文档容器组 (3D效果)
│   ├── 文档1 (最上层)
│   ├── 文档2 (中间层)
│   ├── 文档3 (底层)
│   └── 重叠布局 (1/3重叠)
└── 交互效果
    ├── 聚焦放大 (讲解时)
    └── 轻微晃动 (强调)
```

#### 设计参数 - 苹果风格优化

- **文档宽度**: 70%屏幕 (更大的视觉存在感)
- **3D效果**: 更柔和的角度 (X:3°, Y:6°) + 透视变换
- **材质**: 纸张纹理 + 柔和阴影 (创造真实的文档堆叠感)
- **圆角**: 12px (现代化的圆角设计)
- **阴影**: 渐变阴影 (近大远小)
- **动画**: 苹果的卡片式动画 (slide + rotate, 弹性效果)

### 5. 标题文字模板 (Title Text)

**适用场景**: 章节标题、地点名称等简短信息

#### UI层级结构

```
文字容器 (居中)
├── 大号文字 (屏幕高度30%)
├── 文字阴影 (X2px Y2px)
└── 动画 (淡入0.2s + 淡出0.2s)
```

## 🎨 视觉设计规范 - 苹果风格系统

### 色彩系统 (Apple Color System)

- **主色**: 金色 (#FFD700) - 作为强调色使用，参考苹果的 accent colors
- **背景色**: 动态背景 (浅色/深色模式适配)
- **文字色**: 系统文字色阶 (primary, secondary, tertiary)
- **透明度**: 使用苹果的透明度层级 (80%, 60%, 40%, 20%)

### 字体系统 (Apple Typography)

- **标题字体**: SF Pro Display (或系统无衬线字体)
- **正文字体**: SF Pro Text
- **字重**: 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)
- **行高**: 1.2-1.4 (更好的可读性)
- **字间距**: -0.02em 到 0.01em (根据字体大小调整)

### 动画规范 (Apple Motion)

- **缓动函数**: iOS风格的弹性动画 (spring, ease-out)
- **时长**: 0.2-0.4s (更自然的动画节奏)
- **层级动画**: 不同元素的交错动画创造深度感
- **微交互**: 按钮按下、悬停等细节优化

## 📱 响应式设计

- **桌面**: 1200px+ (完整功能)
- **平板**: 768px-1199px (简化调整面板)
- **手机**: <768px (只读模式，隐藏编辑功能)

## 🔧 交互设计

- **双击编辑**: 文字内容直接编辑
- **拖拽调整**: 位置和大小调整
- **悬停预览**: 参数变化实时预览
- **撤销重做**: 支持操作历史
