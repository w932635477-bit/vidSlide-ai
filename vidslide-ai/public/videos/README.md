# VidSlide AI 演示视频资源

## 视频文件说明

### hero-bg.mp4 / hero-bg.webm
- **用途**: 首页背景视频
- **时长**: 30-60秒循环
- **内容**: 抽象的科技感动画，展示AI处理过程
- **规格**: 1920x1080, H.264/MP4 + VP9/WebM

### demo-loop.mp4
- **用途**: 功能演示循环视频
- **时长**: 30秒循环
- **内容**: VidSlide AI核心功能的快速演示
- **规格**: 960x540, H.264/MP4

## 视频制作指南

### 背景视频要求
1. **风格**: 现代科技感，抽象几何图形
2. **色彩**: 蓝色系 (#007AFF) 渐变
3. **动效**: 平滑的粒子流动，数据流效果
4. **音频**: 无声或者极轻的科技音效

### 演示视频要求
1. **内容**: 展示完整的AI处理流程
2. **字幕**: 关键步骤的文字说明
3. **时长**: 30秒，循环播放
4. **分辨率**: 960x540 (16:9)

## 临时解决方案

当前使用纯色背景代替视频，如需真实视频请联系设计师提供。

## 文件大小优化

- MP4: H.264编码，目标文件大小 < 2MB
- WebM: VP9编码，提供更好的压缩率
- 考虑使用WebP动画作为备选方案
