## 🔧 最终修复步骤

### **步骤1: 修复ECS科学上网**
**目标**: 让3006端口可以访问，启用科学上网

1. **阿里云控制台** → **ECS实例**
2. **停止实例** → **启动实例** (重启)
3. **等待3-5分钟** 让网络配置生效

### **步骤2: 验证科学上网**
```bash
# 启用代理
networksetup -setwebproxystate Wi-Fi on
networksetup -setsecurewebproxystate Wi-Fi on

# 测试访问
curl -x http://127.0.0.1:1081 -I https://vercel.com
```

### **步骤3: 部署到Vercel**
**科学上网修复后**:
1. 访问 https://vercel.com
2. 注册账户 (支付宝支付)
3. 导入VidSlide AI项目
4. 配置AI API密钥
5. 一键部署上线

### **步骤4: 享受完美体验**
- ✅ **Vercel全球CDN** (无需代理)
- ✅ **ECS科学上网** (访问受限服务)
