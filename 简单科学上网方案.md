# 🚀 简单科学上网方案（推荐个人用户）

## 📋 为什么不推荐阿里云企业方案？

阿里云跨域加速方案虽然功能强大，但**不适合个人用户**：

### ❌ 缺点：
- **费用高昂**：SASE + GA + 服务器费用，每月数百元起
- **配置复杂**：需要多个阿里云产品，配置步骤繁多
- **维护困难**：需要专业技术人员维护
- **适用场景**：主要面向跨地域办公的企业

### 方案二：浏览器代理扩展

#### SwitchyOmega (Chrome/Firefox)
1. **安装扩展**：
   - Chrome: https://chrome.google.com/webstore/detail/proxy-switchyomega/pknpgandnjmpccbjhomgfognkdlieoih
   - Firefox: https://addons.mozilla.org/zh-CN/firefox/addon/switchyomega/

2. **配置代理**：
   - 协议: SOCKS5
   - 服务器: 127.0.0.1
   - 端口: 1080

3. **启动V2Ray**：
   ```bash
   cd "/Users/weilei/VidSlide AI"
   ./v2ray -config v2rayu_config.json
   ```

### 方案三：系统代理设置

#### macOS系统代理
```bash
# 设置SOCKS5代理
networksetup -setsocksfirewallproxy "Wi-Fi" 127.0.0.1 1080
networksetup -setsocksfirewallproxystate "Wi-Fi" on

# 设置HTTP代理
networksetup -setwebproxy "Wi-Fi" 127.0.0.1 1081
networksetup -setwebproxystate "Wi-Fi" on
```

## 💡 如果一定要用阿里云方案

阿里云方案适合以下情况：
- 企业用户，有多个办公地点
- 需要安全的内网访问
- 有专业IT团队维护
- 预算充足

### 配置流程：
1. 开通阿里云SASE服务（约¥500/月起）
2. 配置全球加速GA（按量付费）
3. 部署连接器到服务器
4. 配置加速策略
5. 员工下载SASE客户端

**总成本**：首次配置复杂，后续每月数百元

## 🎯 我的建议

**对于个人用户，最佳方案是**：

1. **优先**：找朋友帮忙下载ClashX客户端
2. **备选**：使用浏览器扩展 + 命令行V2Ray
3. **最后**：考虑阿里云企业方案（成本高，配置复杂）

阿里云方案更适合企业环境，而个人用户使用简单的V2Ray客户端就足够了。
