# 🚀 一键科学上网使用指南

## 📋 可选方案（按推荐程度排序）

### 方案一：Shell别名 ⭐⭐⭐⭐⭐ (最推荐)
**优势**：最方便，在任何终端窗口输入简单命令即可

#### 设置方法：
1. 打开终端，输入：
   ```bash
   nano ~/.zshrc
   ```
2. 在文件末尾添加：
   ```bash
   # 科学上网别名
   alias vpn='cd "/Users/weilei/VidSlide AI" && ./一键启动科学上网.sh'
   alias vpnstop='pkill -f v2ray'
   alias vpntest='curl -x socks5://127.0.0.1:1080 https://www.google.com -I | head -1'
   ```
3. 保存退出（Ctrl+X, Y, Enter）
4. 重新加载配置：
   ```bash
   source ~/.zshrc
   ```

#### 使用方法：
```bash
vpn      # 启动科学上网
vpnstop  # 停止科学上网
vpntest  # 测试连接是否正常
```

---

### 方案二：桌面快捷方式 ⭐⭐⭐⭐
**优势**：双击即可启动，无需打开终端

#### 设置方法：
1. 在访达中找到 `科学上网启动器.command` 文件
2. 右键 → **存储到桌面**
3. 或者直接拖拽到桌面

#### 使用方法：
- 双击桌面上的 `科学上网启动器.command` 文件
- 系统会打开终端并自动启动科学上网

---

### 方案三：原生脚本 ⭐⭐⭐
**优势**：直接运行脚本，功能完整

#### 使用方法：
```bash
cd "/Users/weilei/VidSlide AI"
./一键启动科学上网.sh
```

## 🎯 推荐使用流程

### 日常使用（推荐Shell别名）：
1. 打开终端
2. 输入 `vpn` 并回车
3. 等待启动完成
4. 打开浏览器，切换到代理模式

### 临时使用（推荐桌面快捷方式）：
1. 双击桌面快捷方式
2. 等待启动完成
3. 开始上网

## 📱 浏览器配置（一次性设置）

### Chrome + SwitchyOmega:
1. 安装SwitchyOmega扩展
2. 创建新情景模式：
   - 协议：SOCKS5
   - 服务器：127.0.0.1
   - 端口：1080
3. 保存并设为默认

### Firefox + SwitchyOmega:
1. 安装SwitchyOmega扩展
2. 配置同上

## 🛑 停止和清理

### 停止服务：
```bash
vpnstop                    # 使用别名
pkill -f v2ray            # 直接命令
```

### 查看状态：
```bash
ps aux | grep v2ray       # 查看进程
netstat -an | grep 1080   # 查看端口
```

## ⚠️ 注意事项

1. **权限问题**：首次运行可能需要输入管理员密码
2. **端口占用**：确保1080端口未被其他程序占用
3. **网络环境**：确保可以访问V2Ray服务器
4. **系统代理**：脚本会自动设置系统代理，完成后可手动关闭
